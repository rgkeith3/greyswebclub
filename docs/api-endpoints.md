## API Endpoints
-----
### HTML API
#### Root
- GET / -loads React web app

### JSON API
#### users
- POST /api/users
- PATCH /api/users

#### session
- POST /api/session
- DELETE /api/session

#### posts
- GET /api/posts
- POST /api/posts
- GET /api/posts/:id
- PATCH /api/posts/:id
- DELETE /api/posts/:id
